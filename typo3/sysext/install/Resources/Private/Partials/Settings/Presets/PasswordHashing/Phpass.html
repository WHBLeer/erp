In almost all cases, a modern hash algorithm like Argon2i should be preferred and is good to go. phpass is a portable public domain password hashing framework for use in PHP applications since 2005. The implementation should work on almost all PHP builds. It might be a suitable password storage hash method in seldom cases if third party systems must use the same password hash on a low database level and no sane different authentication service can be used for whatever reason. That's funny: phpass is always available!
